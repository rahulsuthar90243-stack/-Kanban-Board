# Kanban Board

A simple and interactive **Kanban Board** built using **HTML, CSS, and JavaScript** to manage tasks across different stages of a workflow.

The application allows users to create tasks and move them between columns using **drag-and-drop functionality**, making task management simple and visual.

---

## 🚀 Features

- Drag & Drop tasks between columns
- Task organization using **Todo, Progress, and Done** sections
- Dynamic task counting in each column
- Data persistence using **LocalStorage**
- Clean and minimal user interface

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Drag and Drop API  
- LocalStorage
